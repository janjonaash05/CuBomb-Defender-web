About Mechanics

Game Mechanics

Description

Bomb

- Enemy object

- Floats towards the player's Shield

- Can be one of the 8 colors

- Can be shot down manually or automatically (via Turret Control)

- Player must have the correct colors set on their Turrets

- If not shot down in time, removes either 1 Shield charge or deals 1 damage to the Core

- Stationary Friendly objects

- Each one corresponds to 1 Turret and has 2 functions:

- Set the color on the Turret (color buttons)

- Order the Turret to shoot all bombs with the matching color (auto button)

- Stationary Friendly objects

- Each one can have 1 of 4 colors (according to the Turret Control)

- Manually shoot down Bombs by clicking on them

- Each manually shot down Bomb heals the Core

- Each automatic use from Turret Control removes one charge

- If all charges are depleted, they recharge back over time

- Automatic use is disabled while recharging

- Stationary Friendly object

- 2 functions: switch between modes and turn the Slider on/off

- 2 modes: Bolt and Full-Auto

- Bolt provides the Slider with one powerful projectile

- Full-Auto provides the Slider with a magazine of weak projectiles

- If projectile(s) are depleted, they recharge back over time

- Mode is disabled while recharging

- Stationary Friendly object

- Click or hold to shoot Slider Controls projectile when on

- Aim towards cursor when on

- The scope glows when successfuly aiming at a target

- Enemy object

- Target for Slider: destroyed in 1 shot from Bolt and multiple shots from Full-Auto

- Rises up and begins charge-up process

- During charge-up, it moves around and adds color bars

- If all bars are colored, it aims at Turret Controls and launches its projectiles

- If not destroyed before they reach Turret Controls, they disable them and Turrets for a set time

- Enemy objects

- Stars - 1-8 at a time

- Stars - Target for Slider: destroyed in 1 shot from both Bolt and Full-Auto

- Stars - Emerge, then begin charge-up process

- Stars - After charge-up, they fall towards a certain point

- Supernova - Grows from each fallen Star

- Supernova - After all Stars fall, it begins charging-up by adding all fallen Star colors

- Supernova - When charged-up, it directly blasts the Core with lasers, 1 damage per Star fallen

- Friendly object

- Consists of 3 parts: Emitter, Storage, Adapter

- Emitter - Emits shield and blocks incoming Bombs from damaging the Core

- Storage - Stores shield charges, each hit blocked removes one charge

- Storage - If all charges are depleted, they recharge over time and all damage goes to the Core

- Adapter - Adapts energy from the Core into Storage recharge speed

- Friendly object

- Consists of 2 parts: Core and Core Ring

- Core - Provides energy to the Shield's Adapter based on its health

- Core - 6 stages of health, 17 health points

- Core - Health can be replenished by manually shooting Bombs with a Turret

- Core Ring - Displays health by colored segements

- Enemy object

- Increases its charge-up value with each damage to the Core

- After Core death, it begins a charge-up process that causes a GAME OVER when finished

- This can be prevented by manually shooting a manually shooting Bomb with a Turret
